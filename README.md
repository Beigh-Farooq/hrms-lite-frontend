# HRMS Lite – Frontend

## Project Description
HRMS Lite Frontend is a web-based admin interface for managing employees and tracking attendance.
It provides a clean, professional, and responsive UI built using Material UI.

---

## Tech Stack Used
- JavaScript
- React (Vite)
- Material UI (MUI)
- Axios
- HTML / CSS

---

## Features Implemented
- Employee management UI
- Attendance tracking UI
- Responsive layout
- Tabs-based navigation
- Loading, empty, and error states
- Integrated with live backend API

---

## Prerequisites
Ensure you have the following installed:

- Node.js (v18+ recommended)
- npm or yarn
- Git

---

## Local Setup Instructions

### 1️ Clone the repository
    git clone https://github.com/<your-username>/hrms-lite-frontend.git
    cd hrms-lite-frontend

### 2️ Install dependencies
    npm install

### 3️ Configure environment variables
    Create a .env file in the project root:
    VITE_API_BASE_URL=http://127.0.0.1:8000

### 4️ Run the application
    npm run dev
